## :toolbox: Getting Started

### :running: Run Locally

Clone the project

```bash
  git clone git@github.com:Behzadsharafi/React-eShop.git
```

Go to the project directory

```bash
  cd React-eShop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
